Tags Name Description

<tbody></tbody> Table Body The table body conatins all the tables data.
<tr></tr> Table Rows Adds rows to the table.
<td></td> Table Data Adds cells in the rows to hold information.
<tfoot></tfoot> Table Footer The footer contains results of the data in the table body.
